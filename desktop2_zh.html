v2.2.19 Beta 版（全新桌面版）

終於，再一次，很高興見到你。

呈現在您眼前的正是全新的純純寫作桌面版。

全新的純純寫作桌面版是一個巨大工程量的產品，至少需要五年以上的時間才能完成，如今發佈的本客戶端實際上是一個半成品。目前許多功能和 UI 都還未實現或處於初始態，大量的細節缺乏打磨和調整……

由於許多用戶熱切希望儘快見到它，所以即使距離真正成熟的版本還很遠，但我們決定發佈它了，或許這是一個不錯的開端，它已經比 2017 年初次發佈的純純寫作 Android 版來得新進和豐富了，而且已經支持雲同步了，不必要和手機實時連接。

前路漫漫，百廢待興，艱難且漫長，但我們沒有放棄，馬不停蹄，相信我們終有一天能抵達那個令人滿意且最超前的里程碑。在此之前，全新的純純寫作桌面端將長久保持完全免費。

希望您能爲一些細節感到欣喜，同時給予我們一些理解和更多等待時間，將不勝感激，不懈努力。

當前版本可能會有許多不足和問題，請多擔待！

最後，爲了避免一些二道打包販子在安裝包裏插入風險代碼，請確保您從 https://writer.drakeet.com/desktop2 下載和更新本軟件。目前它是完全免費的，也無需破解版，請勿輕信任何破解版。

如果您是 Windows 系統用戶，請下載安裝 PureWriter2-Windows-x64.exe，目前暫時僅支持 Windows 10/11

macOS 則使用 PureWriter2-macOS.dmg，下載後右鍵點擊該文件選擇“打開”即可進行安裝。

⚠️ Windows 系統可能遇到殺毒軟件誤報，請放心並將桌面版安裝包及安裝好的文件（C:\Program Files\Pure Writer 2）加入白名單即可。我們是實名註冊的開發者和公司，不可能做病毒軟件的，那樣不僅身敗名裂而且要進監獄、破產。Windows 上的殺毒軟件可能對於沒有簽名的 exe 默認報毒，我們後續將解決此問題。

PureWriter2-2.2.19-macOS-aarch64.dmg 用於蘋果 M 系列芯片 / Apple M Silicon

PureWriter2-2.2.19-macOS-intel.dmg 用於 Intel 芯片

⚠️ 若更新，請關閉本軟件後再安裝更新，這樣可以避免「需要重啓」問題。

⚠️ 請注意，OneDrive 和「阿里雲盤」都不是堅果雲。目前必須在純純寫作 Android 端切換到與桌面端同一個雲備份/雲同步網盤方可相互同步：

（因爲雲盤只是一個單向備份的載體，或者雲同步的中轉，所以沒必要遷移雲盤數據，直接切換即可）

 

⚠️ 關於 Windows v2.1.9 版本的注意事項：

在安裝 v2.2.19 版本之前，請先卸載 v2.1.9 版本！

卸載方式：右鍵點擊「開始」，選擇「安裝的應用」或「程序與功能」，然後在軟件列表中找到「Pure Writer 2」v2.1.9 版本，點擊「...」、「卸載」🙏

 

v2.2.19 版本要點如下：

  • 修復 如果 Windows 系統中沒有制定默認瀏覽器，那麼阿里雲盤授權頁面無法打開，進而導致無法授權且遇到 404 錯誤

  • 修復 在部分 Windows 屏幕上兩側留白過大，導致一行內容很短

  • 修復 使用「覆蓋」進行恢復時，當前文章沒有刷新且可能遇到 db 文件被鎖定的問題

  • 加深 白色主題字體

  • 修復 在 Windows 上如果瀏覽器是由純純寫作打開的那麼關閉純純寫作後瀏覽器會被一併關閉

  • 實現 對於「每日統計」的支持，之前這方面是廢的，會導致手機上統計不準

  • 支持 同步手機上的字數統計模式

  • 優化 同名文章顯示書名

  • 修復一些彈窗報錯

  • 修復 重影問題

  • 加入 霞鶩文楷

  • 支持 自動更新，以後一般情況下不需要再手動下載、安裝更新了！

  • 修復「OneDrive」點擊登出時軟件無響應

  • 改善「雲同步」同步頻次

  • 適配「微信輸入法」符號自動配對

  • 支持「阿里雲盤」

  • 支持雲備份和雲同步，但目前僅限 OneDrive 和「阿里雲盤」，您必須在 Android 端上也登錄同一個網盤才能與之同步，請在正式使用本軟件之前，設置好雲備份網盤、開啓雲同步，看看這方面是否正常，因爲 OneDrive 是國外微軟公司的服務，在部分國家和地區可能無法順暢連接 OneDrive，若使用 OneDrive 請首先檢查此方面

  • 支持 ESC 鍵打開文章和書籍搜索，但目前還不能通過上下方向鍵在搜索結果中移動焦點條目

  • 支持 ctrl + f 或 ⌘ + f 查找某關鍵詞，也可以選定一個詞語後，再按此快捷鍵對其進行更多查找

  • 支持靜態光標

  • 暫不支持摺疊卷

  • 暫不支持歷史記錄，但支持 ctrl + z 或 ⌘ + z 撤銷

  • 支持在右側側欄打開文章

  • 不支持 Markdown

  • 不支持每日統計，並且會造成 Android 端當天的每日統計不準確（將不會統計在桌面端書寫的內容）

    ……
