>#### v3.1.2 (all-new desktop version · currently completely free)

The all-new Pure Writer Desktop is a very large undertaking, one that will take at least five years to complete. The client being released now is still, in practice, a Beta version. Many features and parts of the UI are not yet implemented, or are still in an early state, and many details still need careful polishing and adjustment…

Many users have been eager to see it as soon as possible, so even though it is still far from a truly mature version, we decided to release it. Perhaps this is a good beginning: it is already more modern and richer than Pure Writer for Android was when it first launched in 2017, and it already supports cloud sync, so it does not need to stay connected to your phone in real time.

There is a long road ahead and much remains to be rebuilt. It will be difficult and slow, but we have not given up; we keep moving, and we believe that one day we will reach a milestone we can truly be satisfied with. Until then, the all-new Pure Writer Desktop will remain completely free for a long time. That is why it currently does not require you to sign in with a Pure Writer Pro account; data is synced through your cloud backup drive.

We hope some of its details will make you happy, and we would be deeply grateful for your understanding and for more time as we continue working on it.

The current version may still have many shortcomings and issues. Thank you for bearing with us.

Finally, to avoid repackaged installers that may have risky code inserted by third parties, please make sure you download and update this app only from https://writer.drakeet.com/desktop2. It is currently completely free, so please do not trust any cracked version.
