Pure Writer Q&A

Different kinds of word count

In the Chinese environment, Pure Writer has three modes of word count: without spaces and line breaks, with all symbols and text, and excluding all punctuation, spaces and line breaks. The scope of the statistics can be clearly understood based on the literal content, so we will not elaborate on it here.

Under English usage, if you find your word count will count a letter as a word instead of a word as a count, you can turn on "Treat a word as a count" in the setting item to use word count, but the word count option for Chinese will be disabled and will be defaulted to "Do not include spaces and newlines".

About the use of the landscape screen

At present, Pure Writer is not very compatible with cell phones in landscape, so if you want to use it in landscape, it is most recommended to use a tablet or computer Android system.

When using Pure Writer on a large screen device, you can open the Tablet Settings and Guidelines on the right side of the top bar, through which you can turn on or off some optimizations made for large screen devices.

Shortcut keys for shortcut entries: Press and hold the Ctrl or Alt key on the physical keyboard to shortcut the first 10 shortcut entries.

The way to open the landscape screen: Turn off the system's Orientation Lock and rotate your device to landscape.

If you can't find this setting, please open your phone's Settings, find the setting with the word Display or Display & Brightness in it, find Turn off Vertical Screen Lock or Auto Rotate Screen, and turn it on.

You can also make it a landscape display by dragging and dropping the border when using the tablet's desktop mode or PC Android.

Introduction to the use of shortcut input fields

The shortcut input field is a small column above the Pure Writer keyboard. In this column, users can set their own shortcut input content (such as time and date, people's names or common punctuation marks, etc.) or shortcut tools.

Click "⊕" or "⊕ Create Shortcut" to create a new shortcut or shortcut tool, and long press an existing shortcut to sort, modify or delete it. Move the cursor position in "Content" to set the cursor's landing point after inserting this content.

The Formatting Options menu opens when you click on the ⌘ symbol (M↓ icon in Markdown) on the left side of the shortcut input field (opens as a shortcut input panel in Markdown).

It is worth noting that, unlike elsewhere, when you select a paragraph and insert a shortcut entry with the cursor landing in the middle of the text, this item appears on both sides of the selected text instead of replacing the paragraph; when writing, this way you can quickly add symbols to sentences or paragraphs that are missing quotation marks or brackets.

The Quick Input field is supported to be turned off, you can turn off the Quick Input field in Minority Options in Settings.

About Quick Tools

The "Create Shortcut Input Item" allows you to create shortcut tools located in the shortcut input field. Currently, the supported shortcut tools are Redo, Indent at the beginning of a paragraph, Formatting option for one-click typesetting, Focus to end of text, Focus to top, Copy currently selected text or full text, Search article/Quick jump to an article, Quick move cursor (need to be enabled separately in Minority option). When creating or modifying a shortcut input item, users can set the function of that shortcut button by themselves.

In subsequent updates to Pure Writer, several optimizations and adjustments may still be made to the shortcut input field, so you can make suggestions to the developers if you are not satisfied with the existing features.

About shortcut input item cursor landing point

The shortcut entry consists of two parts: "Content" and "Short Description". The cursor position of the shortcut entry depends on the cursor position in the "Content" column.

If "|" is in Contents', the cursor will fall between"after clicking the quick input item, and if"|"is inContents', the cursor will fall after " after clicking the quick input item.

How can I insert the current time through the shortcut input item?

To set this item, you need to use the format ${<format>} and add the corresponding content in the Content of the Quick Input Item.

First put a table of quick time input, if you can't read it, you can skip it and directly use the million dollar formula given later.

A few golden formulas are included, which can be copied and added or deleted according to your needs.

12-hour timer: ${GG yyyy-MM-dd E aa hh:mm:ss.SSS(z)}

Output format is : Era Year-Month-Day Week Morning/Afternoon 12 Hour Timer:Minute:Second. thousandths of a second (time zone information)

24-hour timer: ${GG yyyy-MM-dd E HH:mm:ss.SSS(z)}

Output format is : Era Year-Month-Day Week 24 Hour Timer:Minute:Second. thousandths of a second (time zone information)

You can also directly use Chinese characters if you want to output Chinese character content, e.g.

12-hour timekeeping: ${GG chronological year yyyy year M month d day E aa hh hour mm minute ss second SSS millisecond Time zone: z}

24-hour timekeeping: ${GG chronological year yyyy year MM month dd day EEEEE HH hour mm minute ss second SSS millisecond Time zone: z}

Note: When a single letter is used, "04" will be displayed as "4", and similarly, "Tuesday" will be displayed as "Tuesday Note: When a single letter is used, "04" will be displayed as "4" and "Tuesday" will be displayed as "Tuesday. **

Introduction and problems with status bar focus mode

Status bar focus mode is a feature used to hide most of the notification icons in the status bar.

It is located to the right of the paragraph spacing switch in the menu of "Line Spacing|Left and Right Margins". Xiaomi/Redmi phone users can't hide the status bar icon because of the limitation of MIUI, so the icon is hidden by default.

Please understand that this feature has different degrees of adaptation on different models, and the effect may also vary depending on the ROM, so please understand that this mode cannot be set successfully on some models.

About formatting options

Formatting options are options used to quickly give uniform formatting to the content of the full text.

In it, you can add or remove extra blank lines between paragraphs, insert spaces between English and Chinese for easier reading, add indentation or remove spaces at the beginning of paragraphs, and quickly traditionalize or simplify the whole text.

The formatting option is useful for editing long paragraphs, so don't feel intimidated by the word "formatting" and feel free to use it.

How to find/replace text?

Click the More icon in the top bar on the main page and open Find & Replace (Find & Replace) in the pop-up menu.

The top input field is the content you want to find, and the bottom input field is the content you want to replace. If you only need to find, you do not need to enter the content in the lower column; when you want to replace, you need to select the part you want to replace and click Replace.

The first column of the selection bar below the input box allows you to choose different search modes and ranges, currently provided are: "Current Chapter", "Current Book", "All Books", and "The second column is for some function buttons, such as: "Replace", "Select All", "Inverse ", "Put away soft keyboard" and the switch of "Search article/Quick jump to an article" panel.

For article search, please use the "Search Article/Quick Jump to an Article" function.

How to use the "Search Article/Quick Jump to an Article" feature?

The main purpose of this feature is to search for articles with specific content in books, not relevant content within articles, so even if multiple keywords appear in a single article, only the first keyword in the text will be displayed as a search result.

Simple usage: Enter the relevant content from that article in your memory to search for articles with that content. When there are multiple keywords/phrases, you can search by line feed entering multiple keywords/phrases, but only the first keyword/phrase will be highlighted in its results.

This search method can be used for chapter name/title search, where you can enter the title of a relevant article to search for articles with that content. Although the "title only" feature is not currently supported, it is possible to search for special characters (such as spaces, "chapter") used in the title together to achieve similar results.

Regular expressions (regular expressions): you can use regular expressions to search the content of the article, as regular expressions users are less, here do not provide tutorials, interested can check on the network.

Is there a word limit for individual chapters?

Pure Writer completely overcomes the technical challenge of having a limit on the number of words in a single chapter, enabling unlimited word count in a single chapter, with the number of words in a single chapter determined only by your personal wishes and the performance of your phone.

Pure Writer has been tested to be able to browse articles of ten million words without lagging, and theoretically to browse 100 million words without lagging (this data is obtained from the developer's experiments. Please note that random experiments may cause the cloud backup traffic to be quickly exhausted).

In actual use if you want to edit, it is recommended to keep the word count of a single chapter below 20,000 to get a smooth experience. Because due to cell phone performance limitations, some devices will have a slight lag when editing after a single chapter word count of more than 10,000, browsing will not be affected.

How can I check the creation/update time of a chapter?

  • Click the "More" icon at the top of the chapter list sidebar, and click "Zoom".
  • Click the "Sort Rules" icon at the top of the chapter list sidebar, click "Show Time Type" and select the desired display type.

How do I open an external document?

Pure Writer supports opening external TXT/Markdown documents.

Generally you can find "Open with Pure Writer" from "Open with Third Party Applications" in the system file manager, some managers may not show "Open with Pure Writer", you can replace other file managers to open it.

Why can't I open/import documents that are too large?

Due to the performance limitations of cell phones, Pure Writer will lag when opening some large TXT documents, which makes many users mistakenly believe that the application or the system is stuck. Therefore, Pure Writer actively disables users from importing documents that are too large, and users can still import them by copying and pasting.

Large documents are typically 300KiB or larger, and vary depending on the performance of the phone.

How to change the font and font size?

Custom fonts are a premium feature, but free users can experience the feature for free.

Android 11 or below: Place the TTF file of your favorite font in the /Documents/PureWriter/App folder of the storage directory, open Custom Fonts in the settings, and then exit Pure Writer and re-enter to load the custom font.

Android 11 and above system settings: open "Custom Fonts" in Settings, tap "Select Fonts", and select the preferred font in the pop-up interface.

There is an option to change the font size of the title and body text in the settings.

How to make text translucent?

Click the "More" icon on the top bar of the homepage, and click "Line Spacing|Left and Right Margins" in the pop-up menu below to modify the transparency of the text.

How do I adjust the left and right margins of the text?

Click the "More" icon on the top bar of the home page, and click "Line Spacing|Left and Right Margins" in the pop-up menu below to modify the left and right margins of the text.

How do I adjust the line/paragraph spacing?

Click the "More" icon on the top bar of the home page, and click "Line Spacing|Left and Right Margins" in the pop-up menu below to modify the line spacing and paragraph spacing of the text.

Why can't the text be aligned at both ends?

Android's native TextView can't align text at both ends, only the WebView can, but the WebView's performance is poor, so Pure Writer chooses to use the TextView, which has both performance and alignment.

This issue is not useful to give feedback to Pure, Pure Writer can't fix the problem on Android, please give feedback to Google official about the issue.

What if the font color is not very obvious after modifying the background?

The font color (black and white only) can be forced to be changed under the "Fixed, No Automatic Switching" option in Theme & Drafting Paper.

Can Pure Writer change the font color?

Currently Pure Writer only supports forcing black and white fonts to be switched in Themes and Scripts, no font colors can be modified, and there are no plans to support related features.

You can use Markdown to implement related functions if you need.

How do I redo content that has been undone by mistake?

Click on the More icon in the top bar and there is a Redo button at the bottom of the pop-up menu.

After a single click to redo, the settings menu will disappear, providing enough visibility for the user to click multiple times to redo.

It is also possible to perform quicker undo and redo operations through the Quick Toolbar, which is available by creating the relevant tools in the Quick Input field.

How to restore the historical version of an article by history?

In the "More" menu on the top bar of the application home page, click "History" to enter the history page.

In the History page, you can go through and preview the recent editing history of the current article, and also restore the current article to the corresponding historical version.

What's wrong with highlighting only one paragraph/line of text when coding?

You can turn it off in the menu of "More" in the top bar of the main page.

Can I export .docx or .epub format?

Pure Writer does not support exporting files in other formats at this time. There are plans to support exporting .docx files, but this is a low priority; exporting formats such as .epub involves too many changes to be discussed later.

exported documents in the reader to open is garbled how to do?

The encoding format of Pure Writer is UTF-8, some readers may not support or recognize the document as UTF-8 encoding, if the text is displayed as garbled after opening, you can manually switch the text encoding or change other readers to open it.

Why does the list of books pop up when I click on Outline in Time Machine?

Since the outline no longer exists, the purpose of opening the book list is to remind you that you can create a book as an outline and open it in the second editor to check the outline while writing.

What is the typewriter mode?

In typewriter mode, only the paragraph currently being edited will be highlighted, allowing the user to focus more on the text at hand, and also providing a degree of protection from prying eyes from behind.

If typewriter mode is turned on by mistake, you can turn it off in the menu of "More" in the top bar of the main page.

What is read-only mode?

Read-only mode can be approximated as a reading mode, in which you can swipe and flip through your articles at will without worrying about accidental text deletion, missing and other problems brought about by accidental touch, ensuring that the content is safe and risk-free.

Advanced read-only mode feature profile.

(a) When the cursor is not visible, a double-click on the editor is required to add the cursor.

(a) When the cursor is visible, it can be added by simply clicking on the editor.

When the cursor is visible and the input method panel has been retracted, you can tap Back to hide the cursor and re-enter the read-only mode.

How do I turn on read-only mode?

Read-only mode is collected in the ⊕ menu at the bottom right corner. Click or long-press (depending on your settings) the ⊕ symbol at the bottom right corner to enable read-only mode in the pop-up list.

How do I exit read-only mode?

To temporarily leave read-only mode for editing, double-click the editor to break read-only and insert the cursor.

To turn off read-only mode, click or long press (depending on your settings) on the ⊕ symbol in the lower right corner to turn off read-only mode in the pop-up menu.

How can I insert a new post in another location?

Inserting new articles in other locations (at the bottom of the article list, at the end of the volume, after the current article) is collected uniformly in the ⊕ position at the bottom right corner, long press the ⊕ symbol at the bottom right corner to bring up the list of this item.

About The Minority Option

The Minority option offers some options that are relatively more rarely chosen or not recommended by developers to be turned on, such as turning off the smooth cursor, turning off title centering, hiding the shortcut input field, etc.

What should I do if there is an unusual back-color in use?

This problem is usually found in the systems of Chinese cell phone manufacturers. This problem is caused by the forced inverse color of the dark color mode of Chinese systems, and currently Pure Writer cannot completely circumvent this problem.

Solutions that can be provided at this time.

  • Turn off the system's forced dark color for Pure Writer in the phone's system settings.
  • Try to avoid turning on dark color mode when using Pure Writer.
  • Suggest to the relevant system vendor to modify and optimize the relevant functions, and ask the vendor to open the interface for rejecting the forced dark color mode by the system, etc.

Why does the "This feature is an advanced feature" pop-up appear?

Custom manuscript paper, preview (including Markdown quick preview), export, Material Design 2 theme, find and replace, merge into book export, second editor, and custom fonts are advanced features that are available to paid users with unlimited access.

Some of the features allow free users to try, but in the trial will pop-up "this feature is a premium feature" pop-up or half-screen prompt, does not affect the normal use of this feature.

If you are already a paid user and still get this pop-up, please go to the Premium page to check if your Pure Writer account or Google account is logged in, if not, please log in again or go to the Google Store to verify your Google account.

Can I use Pure Writer without purchasing the premium version?

You can, Pure Writer's core features are free, including an excellent editor experience with foolproof cloud backup. You can use most of the features in Pure Writer for free, and Pure Writer does not gain any other revenue by inserting ads for users of the free version.

Every year, there will be price reductions on the double 11, if you are not in a hurry, you can wait for the double 11 event or other events from time to time.

Why doesn't the custom draft paper disappear?

In Pure Writer's Themes and Scripts, the app automatically remembers the scripts the user has used on the selection page to facilitate quick changes to the script.

Pure Writer allows two custom sheets to be memorized at the same time. When the user sets a new sheet, the old one is overwritten and can be deleted by long-pressing on the Once item.

When restoring a backup, does it restore the manuscript paper?

Pure Writer's backup package automatically backs up the bed link or RGB color value of the user's sheet. If RGB color values or bed links are used when setting up the sheet, the original sheet will be restored when restoring the backup; if a local image is used as the sheet, it cannot be restored on other devices.

What's wrong with the indent width?

There are generally two possible reasons for problems with the indentation display: font changes and application (system) language other than Chinese.

  • The most common situation is the problem caused by the modification of the font, the common performance of the display of the indentation length is not enough (generally half-space, that is, the indentation of two full-space only a Chinese character character width) or display as the wrong symbol, dot, box, etc..

    Pure Writer's indent content is two full-cornered spaces, while some fonts are made with full-cornered spaces (U+3000) as the width of half-cornered spaces or other content, resulting in the displayed indent content not being full-cornered spaces but other characters.

    Solution: This situation can only be solved by switching fonts with normal full-space display, please switch the system font or refer to the content in this Q&A to switch third-party fonts; users with technical conditions can modify the full-space in the font to make it conform to the specification.

  • Another situation is caused by modifying the application or system language.

    Pure Writer is released for users all over the world, and its indentation rules will be based on the usage habits of users in different regions, such as four half-space indents in English environment, one full-space indent in Japanese and Korean environment, etc.

    Solution: Switch the application language or system language to Chinese.

Why is there no more indentation when coding?

  • You may have Markdown mode turned on, which will automatically stop indenting and return to normal when Markdown is turned off.
  • If Markdown is not turned on, go to Settings and check to see if Indenting is turned off.

Why is the format .md when exporting text?

The .md format is the export format for Markdown documents.

To export a TXT text document, go back to the main page and click the M↓ icon on the right side of the top bar to turn off Markdown mode.

Can I use Markdown/LaTeX in Pure Writer?

Pure Writer supports the use of Markdown and some HTML tags (such as the underlined <u>text</u>), while LaTeX supports the insertion of interlinear and intralinear formulas, but currently the LaTeX part of the text is somewhat sunken.

To enable Markdown/LaTeX, tap the TXT button on the right side of the top bar to change it to M↓.

To get a preview when using Markdown, click the icon next to the plus sign in the bottom right corner to toggle the preview status.

Markdown shortcut input page

The Markdown shortcut page integrates some common Markdown symbols, so Markdown writers can use it to type Markdown symbols more easily.

In Markdown mode, click the M↓ icon at the far left of the shortcut column to open the Markdown shortcut page; in non-Markdown mode, you need to open the formatting options and move to the left to the Markdown shortcut page.

Here you can quickly type in some of the Markdown tags, such as bold, italic, insert images, etc., though this page's stuff does not support modifications.

Can Markdown be rendered in real time in Pure Writer?

Pure Writer has tried to support Markdown real-time rendering, but after testing, it was found that the performance of today's mobile devices does not yet meet expectations, so Pure Writer will not offer this feature until the performance of mobile devices is up to par.

What should I do if the application is slow to load or takes a long time to open?

This usually happens because you have opened too long articles in Pure Writer. Due to the limited performance of the phone, there is a certain waiting time when opening very long articles.

Pure Writer is an editor, not a reader, so please try to avoid opening excessively long articles with Pure Writer. If you find that the start-up is slow due to excessively long articles, please completely delete the externally imported oversized articles.

And if you support rich text or images, it will make the complexity of the editor rises significantly, complexity means that it is easy to make mistakes and difficult to achieve perfection, focus to do the best.

So I'm sorry, Pure Writer has not planned to support rich text from the beginning, and will not plan to do so in the future.

Why does Pure Writer prompt me to "skip" or automatically create a new article when I open it?

This situation is usually caused by the use of a startup skip application that requires secondary/access privileges. To avoid this, close the application in question or set Pure Writer to skip whitelist.

About the accessibility permission, it is actually very dangerous and will seriously affect the system performance. It is recommended that ordinary users try to avoid using applications (or services) that require accessibility permissions. Accessibility services are essentially designed to serve users with inconvenient operations such as the blind, and abuse of this feature may lead to loss of phone performance or even loss of personal information.

Is there a limit to the number of devices I can log into my Pure Writer account?

Currently a Pure Writer account can only be logged in on a maximum of 3 devices at the same time. If you log in on the 4th device, the earliest previously logged in device will be logged out.

Why do Google Play purchased memberships sometimes lose their activation?

Pure Writer Markdown User's Guide

What is Markdown?

Markdown is a markup language for expressing rich text content in plain text. It is easy to learn and Pure Writer supports Markdown(MD). It is easy to learn and Pure Writer supports Markdown(MD), you can switch it to M↓ by clicking TXT on the right side of the top bar to enable Markdown.

Pure Writer provides a Markdown shortcut panel, you can use them by clicking the M↓ icon on the left side of the shortcut bar.

Caution

  1. space is an important component in Markdown and cannot be missing.

  2. The markup used in Markdown is mostly half-corners, so please do not confuse it with full-corners.

  3. Markdown default ** can not use indentation **, there is a need to indent the paragraph please manually insert the indent.

  4. There is no fully unified specification for Markdown, Pure Writer follows and implements CommonMark's Markdown standard, and some third-party Markdown may not work in Pure Writer (e.g. ==Highlighted Text===).

  5. Components such as headers, lists, quoted text, tables, selection boxes, dividers, etc. cannot be used in-line** and must be used and rendered at the beginning of the line.

  6. A Pure Writer tip that you may not necessarily find: After selecting some text, use the shortcut input items or shortcut Markdown tags inside Pure Writer, when the cursor position of the shortcut input is in the insert item (such as inserting brackets and setting the cursor in the brackets after inserting), it will be inserted directly before and after the selected text, instead of replacing the paragraph text.

  7. By default Markdown states that a line must be blank between two paragraphs or two spaces at the end of a paragraph to actually make a line break on rendering, e.g.

    xxxxxxxxxx
    First line

 

Second line

xxxxxxxxxx


   If you turn on `GFM Line Breaks` in the Pure Writer settings page, you only need one line break to get a line break on rendering. Since `GFM Line Breaks` is turned on by default, this tutorial is written with the `GFM Line Breaks` switch turned on by default.


8. Throughout this guide, the vertical tab `|` is used to refer to the cursor point, except for the table section.


## Full text section


### Title


Markdown headings are represented by the well symbol `#` with spaces, and the number of `#`s represents the hierarchy of headings, from less to more, there are six levels.


The text content of the title is filled in after the space. For example.


```md
# First level title


## Secondary Title


### Third level title


#### Level 4 Title


##### Level 5 Title


###### Grade 6 Title

Headings should actually insert the same number of well symbols # before and after (e.g. ## secondary heading ##), which is not required in Pure Writer.

Another heading style is a first-level heading consisting of at least three equal signs = or a second-level heading consisting of at least three short hyphens/minus signs -, which should be inserted in the line below the corresponding heading text.

Note that there must be no blank lines between the text and the short connecting line, otherwise it will be recognized as a split line.

Bold/italic/bold italic

These three things are said together, mainly because the marker symbols they mainly use are the same, they are all asterisks * or underscores _, the difference only lies in the number of symbols.

Italics are composed of one preceded by one followed by two asterisks or underscores: *italic text*, _italic text_

Bold is usually composed of four asterisks or underscores before and after: **bold text**, __bold text__

And bold italics by analogy are composed of the first three followed by three six asterisks or underscores: ***bold italic text***, ___bold italic ____

No underline style shortcut input is provided in Pure Writer, so it is recommended to use asterisks to bold or italicize.

strikethrough/underline

Strikethrough and underline are both text plus lines, so they are put together.

The strikethrough is composed of four wavy lines in the front two and the back two: ~~strikethrough text~~

You can't directly insert underscores in Markdown, so you need to use the in-line HTML tag <u>|</u> to insert them.

Underline effect can be achieved by inserting text in the <u>|</u> tag, example: <u>Underline text</u>

Ordered list

An ordered list is marked in a simpler way, with a number plus a period plus a space: 1. The first point

A line feed automatically completes the next line of the sequence.

Unordered list

There are three ways to use unordered lists, and Pure Writer offers two of the common ones and provides automatic filling for them.

The first is a short hyphen/ minus sign - plus space: - minus sign list

The second one is asterisk * with space: * asterisk list

In addition, Pure Writer does not provide shortcut input but supports the rendering of markers with the plus sign + plus space: + plus list

There is no difference in the practical use of the three.

A point worth noting is that when rendering consecutive unordered lists, lists consisting of the same tokens will be closer together, while lists consisting of different tokens will be further apart, which can be useful in some cases.

Ordered and unordered lists can be nested with each other, and the nesting can be done by adding at least two spaces in front of the list.

Quote

When you need to quote a passage, try the quote marker, which is composed of a greater than sign > plus a space: > Quoted text

Footnotes

Add a footnote name after the text you need to add a footnote to: [^footnote name]. Then add a footnote in an empty line anywhere in the text (usually at the end): [^footnote name]:footnote content, you can insert a footnote.

Note: A space should not be added after the colon : in the footnote.

Pure Writer's footnotes do not open in a small window, but will jump directly to the browser, so this feature is currently less useful.

Corner labels

Markdown does not natively support top and bottom corner markers, but this can be done in Pure Writer via HTML code.

The code for the upper corner is: <sup>|</sup>; the code for the lower corner is: <sub>|</sub>.

Examples: Citation<sup>[1]</sup>, C<sub>2</sub>H<sub>5</sub>OH.

Left/center/right text

Markdown cannot directly center text, so it uses the HTML code <p align="">|</p>, which Pure Writer provides by default as <p align="center">|</p>, but by modifying the corresponding position code, it can support both left and center, left, center, right.

The format of the text position is: <p align="position">Text that needs to be modified in position</p>

Pure Writer** does not work with **<center>|<\center> code centering.

Insert image

Inserting an image is a relatively important feature, and it has the format: ![]().

The first square brackets fill in the image name, while the second brackets fill in the image location. When you are using a web image you should fill in its web location, when you are using a local image you should check or fill in its local location with the image selector.

It is important to note that when using a network image bed, you should pay attention to protect your privacy and choose a trustworthy image bed; when using a local image, the image cannot be moved or deleted, otherwise it will not be displayed properly, and the image cannot be shared and synchronized with Markdown documents.

Insert link

Similar to inserting an image, the format is: []().

Just enter the link description in the front square brackets and the link in the back round brackets, e.g.: [POPO Original Marketplace](https://www.popo.tw/index.html)

Insert table

Pure Writer provides a shortcut to insert a three-column, four-row table.

Tables in Markdown don't have a lot of formatting like excel, just a simple table with a few columns and rows.

The basic components of the table are the vertical tab |, space, colon : and short hyphen/ minus - four components, tab separates each column of the table, space to determine the text content of the table, colon to determine the location of the text in the table, short hyphen is responsible for marking the table header and auxiliary to determine the location of the text in the table.

Example.

xxxxxxxxxx
| left-justified | centered | right-justified |
| :----- | :----: | -----: |
| left | center | right |
| left | center | right |
| left | center | right |

The actual display results are

Points to note.

  1. When inserting a table, the previous row of the table must be empty, otherwise the table will not be rendered successfully.
  2. :--: means the content and title bar are left-aligned, :--: means the content and title bar are centered, and --: means the content and title bar are right-aligned.
  3. The extra spaces between | and - are ignored and do not affect the layout.
  4. Extra spaces between the content and | are ignored, so for aesthetic purposes you can use spaces to align cells in different rows, and the number of - at the table header should be at least one.
  5. The first | and last | of each line can be omitted, so an extra line should be left blank when entering the content after the form to prevent the next line from being considered the content of the form.

Select box

Markdown's checkboxes are made up of a hyphen/minus - and square brackets [], and a space or the letter x is entered in the square brackets to determine if the checkbox is selected. The order of the checkboxes is short hyphen, space, front square bracket, space or letter x, back square bracket, space plus the text to be selected, all of which are indispensable, and the Markdown shortcut in Pure Writer already provides shortcuts to the corresponding checkboxes to avoid typing errors.

Examples: - [ ] unchecked, - [x] checked

Dividing line

The dividing line is made up of at least three short connecting lines/minus signs - that occupy a separate line.

Note that a blank line should be added between the text and the split line below it, otherwise the text on the split line will be recognized as a title.
